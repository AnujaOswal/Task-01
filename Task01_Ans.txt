4.When a URL is typed in the address bar, the following steps are perform by the browser:
i)The browser checks the cache for a DNS record to find the corresponding IP address and the cache memory is checked if the web address entered has already been ran before.
ii)After that a Transfer Control Protocol(TCP) is initiated between client and server.
iii) HTTP request is sent to the respected server.
iv)Then we  We receive a response from the server with the HTTP code.
iv)And finally at last respective rendering engine and browser engine will create web page and display output on browser.
